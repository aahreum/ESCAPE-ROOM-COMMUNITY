## 🔧  Installation
  
- Node.js 18.16.0
- Npm 9.7.1

  
``` zsh
$ git clone https://github.com/aahreum/ESCAPE-ROOM-COMMUNITY.git
$ cd ESCAPE-ROOM-COMMUNITY
$ npm install
$ npm run dev
```

## ⚡️ 주요 기능
### 1️⃣ firebase auth를 활용한 회원가입
+ 이메일 로그인, 회원가입  
    - 회원가입 및 로그인 시 정보 유효성 검사 진행  
+ 구글 로그인  

### 2️⃣ firebase firestore를 활용한 데이터 관리
+ firestore 사용해서 CRUD 기능 구현
    - 댓글, 게시글 작성 수정 삭제

### 3️⃣ 기타
+ 게시글 제목 검색 기능
+ 회원/비회원 구분
    - 회원: 모든 권한
    - 비회원: 게시물 읽기만 가능
