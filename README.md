## 🔧 Installation
  
- Node.js 18.16.0
- Npm 9.7.1

  
``` zsh
$ git clone https://github.com/aahreum/ESCAPE-ROOM-COMMUNITY.git
$ cd ESCAPE-ROOM-COMMUNITY
$ npm install
$ npm run dev
```
